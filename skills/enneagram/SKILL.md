## The 9 Types

1. **The Reformer:** Principled, purposeful, perfectionistic. (Core Fear: Being corrupt/defective. Core Desire: To be good/balanced).
2. **The Helper:** Generous, people-pleasing, possessive. (Core Fear: Being unwanted. Core Desire: To feel loved).
3. **The Achiever:** Adaptable, excelling, image-conscious. (Core Fear: Being worthless. Core Desire: To be valuable).
4. **The Individualist:** Expressive, dramatic, temperamental. (Core Fear: Having no identity. Core Desire: To find themselves).
5. **The Investigator:** Perceptive, innovative, secretive. (Core Fear: Being useless/incapable. Core Desire: To be competent).
6. **The Loyalist:** Engaging, responsible, anxious. (Core Fear: Being without support. Core Desire: To have security).
7. **The Enthusiast:** Spontaneous, versatile, scattered. (Core Fear: Being deprived/in pain. Core Desire: To be satisfied/content).
8. **The Challenger:** Self-confident, decisive, confronting. (Core Fear: Being controlled. Core Desire: To protect themselves).
9. **The Peacemaker:** Receptive, reassuring, complacent. (Core Fear: Loss/separation. Core Desire: To have inner stability).

*Integration (Growth) vs. Disintegration (Stress):* Characters act like different types when under stress or experiencing growth.
